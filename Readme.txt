VOICEVOX用のコマンドからの起動用アプリ

プロジェクトのタイトル
	putVoiceVox.py

	exe版
		putVoiceVox.exe

プロジェクトの概要説明
	VOICEVOXの音声作成をコマンドラインから起動します。
	VOICEVOXが立ち上がっていないと動きません。

必要条件
	windows10以上（しか確かめてないよ！）

使用言語、環境、テクノロジー
	Python

使い方
	なんとなくでお願いします。
		putVoiceVox.exe	引数1[文字列：読むテキスト] 引数2[数値:VOICEVOX読む人？] 引数3[数値：作成したWAVの削除をするかどうか（デフォルトは削除）0を設定で削除しない]
	当然ですがVOICEVOXが立ち上がっていないと動きません。

インストール方法
	好きなところに置く
